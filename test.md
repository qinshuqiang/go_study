# 数据类型

**数据类型的作用：**告诉编译器这个变量应该以多大的内存存储

#### 命名规范

- 字母、下划线、数字组成
- 开头不能以数字开头
- 名字不能是关键字
- 区分大小写

#### 声明变量的注意事项

- 导入包必须要使用
- 函数内的变量声明了必须要使用
- 同一个{}内不能重复声明变量
- 声明的变量默认值为0
- 变量需要声明后才能使用

#### 变量的声明方式

```go

	var a int      //声明变量
	var b int = 10 //声明并初始化变量
	e := 10        //自动推导类型，通过初始化的值确定类型
	var (
		c int
		d float64
	)// 全局声明多个变量
	f, g := 10, 20 ///多重声明变量并且赋值
	_ := 10        //匿名变量
	
```

#### 常量

```go

	const a int = 20   //常量不可修改，不可重复声明
	const a  = 20.1   //类似变量的声明a:=10，不过声明的是常量
	const (
		c int
		d float64
	)// 全局声明多个常量
```
